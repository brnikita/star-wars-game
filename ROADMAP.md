# K-2SO: Дроид-Изгой — Дорожная карта разработки

> Браузерная 3D-игра на Three.js. Жанр — шутер от третьего лица.
> Главный герой — K-2SO, имперский дроид безопасности-перебежчик из фильма «Изгой-один».

---

## Технический стек

| Компонент | Технология |
|-----------|-----------|
| 3D-рендеринг | Three.js |
| Язык | TypeScript |
| Сборка | Vite |
| Физика | cannon-es |
| Звук | Web Audio API |
| Сохранения | IndexedDB |
| UI | HTML/CSS overlay поверх canvas |

---

## Фаза 1: Фундамент и играбельный прототип

> **Цель:** Можно бегать по коридору, стрелять, видеть K-2SO от третьего лица.

### 1.1 Инициализация проекта
- [x] Vite + TypeScript + настройка конфигов
- [x] Установка зависимостей (three, cannon-es)
- [x] Структура папок (`src/core`, `src/entities`, `src/systems`, `src/ui`, `src/levels`, `src/utils`)
- [x] `index.html` с загрузочным экраном и HUD-контейнером

**Файлы:** `package.json`, `tsconfig.json`, `vite.config.ts`, `index.html`

### 1.2 Игровой цикл и рендеринг
- [x] Класс `Game` — инициализация Three.js сцены, камеры, рендерера
- [x] Игровой цикл через `requestAnimationFrame` с фиксированным шагом физики (60 FPS)
- [x] Рендерер: `WebGLRenderer` (антиалиасинг, тонмаппинг ACES, тени PCFSoft)
- [x] Освещение: HemisphereLight + AmbientLight + DirectionalLight
- [ ] Пост-обработка: EffectComposer → Bloom + FXAA

**Файлы:** `src/core/Game.ts`, `src/main.ts`

### 1.3 Модель K-2SO (процедурная геометрия)
- [x] Голова: приплюснутая сфера, лицевая панель, два светящихся глаза (Emissive + PointLight)

- [x] Руки: 3 сегмента (плечо, предплечье, кисть) + шарнирные суставы
- [x] Ноги: 3 сегмента (бедро, голень, стопа) + шарниры
- [x] Материал: `MeshStandardMaterial` — тёмный металл, metalness 0.8, roughness 0.35
- [x] Анимация конечностей при ходьбе/беге (покачивание рук и ног)
- [x] Высота ~2.1 единицы

**Файлы:** `src/entities/K2SO.ts`, `src/entities/Entity.ts`
